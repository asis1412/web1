    bitnami manager 다시 키려면 bitnami에서 manager를 연다.
    127.0.0.1=IP address{internet protocol address}
    전 세계에 있는 모든 컴퓨터가 웹 브라우저에 127.0.0.1 이라고 치는것은 , 웹브라우저가 설치되있는 각자의 컴퓨터의 웹서버이다.

    • http://127.0.0.1/index.html

    웹브라우저가 index.html 파일을 원한다는 뜻
    1. bitnami파일.
    2. apache2파일.
    3. htdocs파일.
    4. htdocs에 있는것 삭제.
    5. 내가 쓰는 파일 web 복붙.

    주소를 입력하는 것과 브라우저에서 파일을 여는 것(차이점)

      질 적인 차이
    • 주소입력=웹브라우저가 웹서버에게 요청,웹서버가 index.html 열어서 웹브라우저에게 전송.
    • http:// = 웹브라우저와 웹서버가 통신할 때 사용하는 통신 규약(Hyper Text Transfer Protocol)
    • 파일을 여는 것=웹브라우저가 직접 읽어서 열어줌(웹서버 관련X)
    • file:// = 파일을 직접 읽어 화면을 열어주는 것

    ※Hyper Text=웹페이지 Transfer=전송하다 Protocol=규약
    ※하지만 서로 다른 컴퓨터에 있는 웹브라우저와 웹서버가 통신을하기위해선 반드시 http사용해야함.
    스마트폰=스마트폰의 탈을 쓴 컴퓨터

CSS

[font]는 1만개가있으면 1만개 전부 바꿔야함 =/= css[color]는 1만개를 한번에 바꿀 수 있음.
css는 [tag]에서의 중복을 제거해줌.
css는 유지와 보수를 편하게해줌.
css는 html로 부터 디자인의 기능을 빼낸 것.

[style]=html의 속성
a {☆}=선택자, ☆=효과 or 선언
[text-decoration: none;]=밑줄 제거, 효과를 지정한 뒤엔 무조건 ;(세미콜론)
[text-decoration:underline;]=(none사용 후)밑줄을 만드는 것
하나의 태그에는 여러 속성 가능.

[font-size:00px]=글자크기조절 [text-align]=글자 위치 정렬
id선택자>class선택자>tag선택자(모두 형태가 같다=가장 마지막 우선순위)
id선택자=단 한번(중복되선 안됨)
css=구체적인 것(id선택자) > 포괄적인 것(tag선택)
큰 그림(tag선택자)부터 그려놓고 수정(id선택자)하는 것 ※align=정렬하다 property=속성 value=종류

[class="~~"]는 그룹으로 묶는 것.
id선택자는 한번만. 중복 불가. 유일한 값.
웹페이지 모든 a를 가르키는 선택자 < id값=active (더 구체적임 so 권한 더 높음)
